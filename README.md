# 血液細胞分類トレーニング

検査技術学科の大学生向けに作成した、血液細胞分類ゲームのプロトタイプです。

## 起動方法

このアプリは `data/cases.json` を読み込むため、ローカルサーバー経由で開きます。

```powershell
python -m http.server 5173
```

ブラウザで次を開きます。

```text
http://localhost:5173/
```

## データ追加

新しい問題は `data/cases.json` の `cases` 配列に追加します。

- `image`: 画像ファイルへの相対パス
- `answer`: 正答ラベル
- `points`: 解説欄に表示する観察ポイント
- `highlights`: 写真上に重ねるハイライト。`x`, `y`, `w`, `h` は画像枠に対するパーセント指定
- `explanation`: 回答後に表示する解説文

分類ボタンを増減したい場合は、同じファイルの `labels` 配列を編集します。
